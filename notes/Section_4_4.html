    Subsection Problem 13

    If \(x \equiv a \pmod{n}\text{,}\) then \(nk=x-a\) from some integer \(k\text{.}\) If \(k\) is even we can write
    \begin{equation*} 2n\cdot \frac{k}{2}=x-a \end{equation*}
    and since \(\frac{k}{2}\) is an integer this gives
    \begin{equation*} x \equiv a \pmod{2n}. \end{equation*}
    If \(k\) is odd subtract \(n\) from both sides of \(nk=x-a\) to obtain:
    \begin{align*} nk-n \amp = x-(a+n)\\ n(k-1) \amp = x-(a+n)\\ 2n\cdot \frac{k-1}{2} \amp = x-(a+n) \end{align*}
    and since \(\frac{k-1}{2}\) is an integer, we have
    \begin{equation*} x \equiv a+n \pmod{2n}. \end{equation*}
