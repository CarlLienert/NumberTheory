    Section Quiz 8

    Subsection Problem 1

    Prove that if \(p\) and \(q\) are odd primes and
    \begin{equation*} q|a^p-1, \end{equation*}
    then either \(q|a-1\) or \(q=2kp+1\) for some integer \(k\text{.}\) Hint: think about the possible values for \(\text{ord}_na.\)
    🔗
    Since \(q|a^p-1,\) we have
    \begin{equation*} a^p \equiv 1 \pmod{q} \end{equation*}
    which means \(\text{ord}_qa|p.\) Thus, \(\text{ord}_qa\) is either \(1\) or \(p\text{.}\)
    🔗
    If \(\text{ord}_qa=1,\) then \(a\equiv \pmod{q}\) so \(q|a-1.\)
    🔗
    If \(\text{ord}_qa=p,\) then \(p|\phi(q)\text{.}\) That is, \(p|q-1\text{.}\) So, \(pk=q-1\text{,}\) or \(q=pk+1.\) But, \(k\) must be even so we can write \(q=2kp+1.\) (slight abuse of notation, the second \(k\) is not the same as the first.)
    🔗
    🔗

    Subsection Problem 2

    Use question 1 to show that if \(p\) is an odd prime, then the prime divisors, \(q\text{,}\) of \(2^p-\) are of the form \(q=2kp+1.\)
    🔗
    Apply problem 1 with \(a=2.\) If \(q\) is prime then \(q|2-1\) isn’t possible, so it must be that the prime divisors of \(2^p-1\) are of the form \(q=2kp+1.\)
    🔗
    🔗

    Subsection Problem 3

    The following table shows the first several values of \(2kp+1\) for \(p=11\text{.}\) Use the table to find the smallest prime divisor of \(2^{11}-1\text{.}\)
    🔗
    I’ll leave out the table. The idea is to:
    1. Generate integers of the form \(2kp+1\text{.}\)
      🔗
      🔗
    2. Determine if the integer is prime. That is if \(2kp+1\) is a candidate divisor of \(2^p-1.\)
      🔗
      🔗
    3. If \(q=2kp+1\) is in fact prime, determine if \(q|2^p-1.\) To do this compute \(2^p \pmod{q}.\)
      🔗
      🔗
    🔗
    For this example \(p=11.\text{.}\) (It isn’t a particulaly intersesting example, but the computation quickly becomes too much for a quiz if \(p\) is much larger.)
    🔗
    With \(p=11\text{,}\) the first integer to consider if \(q=2\cdot1\cdot11+1=23,\) which is prime. So, we compute \(2^{11} \pmod{23}:\)
    \begin{align*} 2^2 \amp \equiv 4 \pmod{23}\\ 2^4 \amp \equiv \left(2^2\right)^2 \equiv 4^2 \equiv 16 \pmod{23} \\ 2^8 \amp \equiv 16 \cdot 16 \equiv (-7)\cdot(-7) \equiv 49 \equiv 3 \pmod{23}\\ 2^{11} \amp \equiv 2^8\cdot 2^2 \cdot 2 \equiv 3 \cdot 4 \cdot 2\equiv 24 \equiv 1 \pmod{23}. \end{align*}
    Thus, our result indicates that \(23\) divides \(2^{11}-1.\)
    🔗
    We were lucky. The first value of \(2kp+1\) was a divisor of \(2^p-1.\) If either \(2kp+1\) isn’t prime, or it doesn’t in fact divide \(2^p-1\text{,}\) then we’d go on to the next value of \(2kp+1\text{.}\) If \(2kp+1\) reaches the size of \(2^p-1\) then it must be that \(2^p-1\) is itself prime.
    🔗
    The point of the process is that, in practice, we’d pick a much larger exponent in \(2^p-1.\) Finding a prime divisor is non-trivial. This process, as tedious as it might seem, is more efficient than simply running through all the primes until you find one that divides \(2^p-1.\)
    🔗
