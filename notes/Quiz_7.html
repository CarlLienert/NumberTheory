    Section Quiz 7

    Subsection Problem 1

    Prove: \(\phi(n)=\frac{n}{2}\) if and only if \(n=2^k\) for some \(k \geq 1. \)
    🔗
    Suppose first that \(n=2^k\) for some \(k \geq 1. \) Then \(\phi(n)=2^{k}-2^{k-1}=2^{k-1}=\frac{n}{2}.\)
    🔗
    Conversely, suppose \(\phi(n)=\frac{n}{2}\text{.}\) Write \(n=2^kN\) where \((N,2)=1.\) Then
    \begin{equation*} \phi(n)=\phi(2^k)\phi(N)=2^{k-1}\phi(N). \end{equation*}
    Since \(\phi(n)=\frac{n}{2}\) we have:
    \begin{equation*} 2^{k-1}\phi(N)=\frac{n}{2}=2^{k-1}N. \end{equation*}
    That is, \(\phi(N)=N\text{.}\) But this is only true if \(N=1.\) Thus, \(n=2^k\) for some integer \(k \geq 1.\)
    🔗
    Note: The key to this problem is the technique of writing \(n=2^kN\) where \(N\) and \(2\) are relatively prime.
    🔗
    🔗

    Subsection Problem 2

    For any integer \(a\text{,}\) show that \(a\) and \(a^{4n+1}\) have the same last digit.
    🔗
    Use Euler’s Theorem to show that \(a \equiv a^{4n+1} \pmod{10}\) if \((a,10)=1.\)
    🔗
    Also, address the case where \((a,10) \neq 1.\) That is, if \(a \equiv 0, 2, 4, 5, 6, 8 \pmod{10}.\) For example, if \(a \equiv 2 \pmod{10}\text{,}\) then \(a^2 \equiv 4 \pmod{10},\) \(a^4 \equiv 6 \pmod{10}\text{.}\) In fact \(a^{4n} \equiv 6 \pmod{10}\) since successive powers of 6 all end in 6. Finally \(a^{4n+1} \equiv 6\cdot 2 \equiv 2 \pmod{10}.\) So, if \(a\) ends in 2, so does \(a^{4n+1}.\)
    🔗
    Note: We used Euler’s Theorem; you have to be careful to take the hypothesis of this theorem into account. The cases \((a,10)\neq 1\) are dealt with using the process we learned when we first encountered modular arithmetic.
    🔗
    🔗

    Subsection Problem 3

    1. If \(n\) is a square-free integer, prove that
      \begin{equation*} \sum_{d|n}\sigma(d)\phi(d)=n^2. \end{equation*}
      🔗
      First argue that the sum is multiplicative. Both \(\sigma(n)\) and \(\phi(n)\) are multiplicative. Then \(\sigma(n)\phi(n)\) is also multiplicative, and as a result the divisor sum is multiplicative.
      🔗
      Then, we only need to check the desired result for prime powers. BUT, since \(n\) is square-free, we only need check \(n=p:\)
      \begin{equation*} \sum_{d|p}\sigma(d)\phi(d)=\sigma(1)\phi(1)+\sigma(p)\phi(p)=1+(p+1)(p-1)=p^2. \end{equation*}
      🔗
      🔗
    2. If \(n\) is a square-free integer, prove that
      \begin{equation*} \sum_{d|n}\sigma(d^{k-1})\phi(d)=n^k \end{equation*}
      for all integers \(k\geq 2.\)
      🔗
      Notice that part (a) is for \(k=2\text{.}\) We’ll prove the result using induction on \(k\text{.}\) The base case, is part (a).
      🔗
      We have seen that the divisor sum is multiplicative, and since \(n\) is square-free, we consider the case \(n=p\text{.}\) We have:
      \begin{align*} \sum_{d|p}\sigma(d^{k})\phi(d) \amp = \sigma(1)\phi(1)+\sigma(p^k)\phi(p)\\ \amp = \sigma(1)\phi(1)+(1+p+p^2+\cdots +p^{k-1})\phi(p)+p^k\phi(p) \\ \amp =\sigma(1)\phi(1) + \sigma(p^{k-1})\phi(p)+p^k\phi(p)\\ \amp = \sum_{d|p} \sigma(d^{k-1})\phi(d) +p^k\phi(p) \\ \amp = p^k +p^k(p-1)=p^{k+1} \end{align*}
      where we used the inductive hypothosis from the third to the fourth line.
      🔗
      🔗
    Notes:
    1. It’s important to manipulate the sums very carefully!
      \begin{equation*} \sum_{d|p}\sigma(d^{k})\phi(d) \end{equation*}
      is NOT equal to
      \begin{equation*} \sum_{d|p}\sigma(d^{k})\cdot \sum_{d|p}\phi(d). \end{equation*}
      🔗
      🔗
    2. The use of the muliplicativity of the divisor sum is crucial! This is one of the main tools in number theory. In the solutions above I left out the step where you piece the case for \(n=p\) back together for the general case.
      🔗
      🔗
    3. Don’t forget tools from long ago: induction in this case.
      🔗
